# Nothing Phone Widget

A small Android home-screen widget project for Nothing Phone.

## What it does

- Shows a large digital time display.
- Shows the current date.
- Uses a monochrome, dot-inspired layout.

## Open in Android Studio

1. Clone this repository.
2. Open the project folder in Android Studio.
3. Let Gradle sync.
4. Run the app on your Nothing Phone.
5. Long-press the home screen, choose Widgets, and add **Nothing Dot Clock**.
