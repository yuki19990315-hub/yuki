# Yuki Car Video

Nothing Phone を中華系ワイヤレス CarPlay / Android ミラーリングアダプターへ映す前提の、横画面動画再生テスト用 Android アプリです。

## できること

- 起動時に `https://m.youtube.com/` を WebView で表示
- URL 入力、または検索ワード入力から YouTube 検索を起動
- YouTube と Amazon Prime Video のワンタップ起動ボタン
- 「外部」ボタンで、今のページをスマホ側の YouTube / Prime Video アプリまたはブラウザで開く
- 「暗く」ボタンで、再生中にスマホ本体の画面だけ暗めにする
- 動画サイトの全画面再生コールバックに対応
- 車載画面で押しやすい横画面固定 UI
- WebView レンダラー優先度、ハードウェアレイヤー、音楽ストリームの音声フォーカスを使った低遅延寄り設定

## スリープとアカウント設定

車載アダプターへミラーリング中に Nothing Phone がスリープすると、映像出力や再生も止まる可能性が高いため、このアプリは起動中に画面をスリープさせない設定にしています。画面焼けや眩しさを減らしたい場合は、アプリ内の「暗く」ボタンで本体画面の輝度を下げてください。

アカウント設定やログインは、接続前にスマホ側の公式 YouTube / Prime Video アプリで済ませておくのが一番楽です。アプリ内の「外部」ボタンでも現在のページを公式アプリまたはブラウザへ渡せます。ただし Android WebView のログインセッションは公式アプリや Chrome と完全には共有されないため、動画サイト側の仕様によっては WebView 内で再ログインが必要になる場合があります。

## 音の遅延について

アプリ側では WebView の描画優先度と音楽ストリームの音声フォーカスを設定しています。実際の遅延はワイヤレスアダプター、車側入力、Bluetooth / Wi-Fi 経路の影響が大きいため、ズレる場合は車側ではなく Nothing Phone 側から音を出す、またはアダプターの低遅延モードを使う構成も試してください。

## ビルド

Android SDK が入っている環境で以下を実行します。

```bash
gradle assembleDebug
```

生成された APK は `app/build/outputs/apk/debug/app-debug.apk` です。
